# ❤️ Heart Health Risk Analyzer

This project is a Python-based health assessment system that evaluates heart health risk using user lifestyle and physiological inputs.

## 🚀 Features
- BMI calculation
- Risk factor analysis (sleep, BP, smoking, alcohol)
- Health scoring system
- Personalized recommendations
- Multi-user support using loops

## 🧠 Project Idea
This is a rule-based decision system that mimics basic healthcare risk assessment.

## 🛠️ Technologies
- Python

## ▶️ How to Run
python main.py
